# 🌐 Portfolio 2025

Ce projet est mon portfolio personnel, conçu pour présenter mes compétences, mes projets et mon parcours en tant que développeuse web. Il a été entièrement développé avec HTML, CSS et JavaScript.

## 🚀 Fonctionnalités
- 🎨 **Design responsive** : Adapté aux écrans de toutes tailles (ordinateurs, tablettes, mobiles)
- 📂 **Section Projets** : Présentation de mes réalisations avec des aperçus interactifs
- 📝 **À propos** : Un espace pour en savoir plus sur moi et mon parcours
- 📧 **Formulaire de contact** : Pour me joindre facilement

## 🛠️ Technologies utilisées
- **HTML** : Structure des pages
- **CSS** : Mise en forme et animations
- **JavaScript** : Interactivité et dynamisme

## ⚙️ Installation
1. Clone le dépôt :
   ```bash
   git clone https://github.com/Emmanuelle-nsossani/portfolio2024.git
   ```
2. Ouvre le dossier du projet :
   ```bash
   cd portfolio
   ```
3. Ouvre le fichier `index.html` dans ton navigateur :
   - Double-clique sur `index.html`
   - Ou utilise un serveur local si nécessaire
